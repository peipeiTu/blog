#### 1. 根据条件统计数据条数：`COUNT()组合IF()`
```bash
SELECT COUNT(IF(条件,条件为真的返回值，条件为假的返回值)) AS m from 表
```
`COUNT()`和`IF()`嵌套使用，一条语句统计某个值的所有情况。

#### 2. `GROUP BY`使用
* 分组通常和聚合函数一起使用，如`SUM(),COUNT()`
* `GROUP BY a,b`：只聚合a和b都相同的记录
